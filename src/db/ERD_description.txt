=== Detailed ERD Description ===

Users Table
===========
- user_id (PK, UUID)
- email (string, unique, not null)
- role (string, default 'user', not null)
- first_name (string)
- last_name (string)
- password (string, not null)
- notes (text)
- last_login (timestamp)
- created_at (timestamp, default to current time)
- updated_at (timestamp, default to current time)

Customers Table
===============
customer_id (PK, UUID)
- first_name (string, not null)
- last_name (string, not null)
- email (CITEXT, not null, unique)
- phone_number (string, not null, unique)
- shipping_address (text)
- shipping_city (string)
- shipping_state (string, 2)
- shipping_zip (string, 9)
- billing_address (text)
- billing_city (string)
- billing_state (string, 2)
- billing_zip (string, 9)
- notes (text)
- created_at (timestamp, default to current time)
- updated_at (timestamp, default to current time)
- deleted_at (timestamp, nullable)

Orders Table
============
- order_id (PK, UUID)
- order_date (timestamp, not null, default to current time)
- order_description (text, not null)
- customer_cost (decimal, 14, 2, not null)
- input_expenses (decimal, 14, 2, default 0)
- taxes_fees (decimal, 14, 2, default 0)
- shipping_cost (decimal, 14, 2, default 0)
- total_write_off (decimal, 14, 2, default 0)
- profit (decimal, 14, 2, default 0)
- notes (text)
- order_status (string, not null, default 'pending')
- payment_status (string, not null, default 'not paid')
- customer_id (FK, UUID, references Customers(customer_id), onDelete CASCADE)
- created_at (timestamp, default to current time)
- updated_at (timestamp, default to current time)
- deleted_at (timestamp, nullable)

Products Table
==============
- product_id (PK, UUID)
- name (string, not null)
- description (text)
- price (decimal, 14, 2, not null)
- stock_quantity (integer, not null, default 0)
- category (string, not null)
- firearm_type (string, not null)
- is_custom (boolean, not null, default false)
- display (boolean, not null, default true)
- created_at (timestamp, default to current time)
- updated_at (timestamp, default to current time)

Categories Table
================
- category_id (PK, UUID)
- category_name (string, not null)
- category_description (string, nullable)
- created_at (timestamp, default to current time)
- updated_at (timestamp, default to current time)

ProductCategories Table
=======================
- product_id (PK, FK, UUID, references Products(product_id), onDelete CASCADE)
- category_id (PK, FK, UUID, references Categories(category_id), onDelete CASCADE)

ProductImages Table
===================
- image_id (PK, UUID)
- product_id (FK, UUID, references Products(product_id), onDelete CASCADE)
- image_url (string, not null)
- created_at (timestamp, default to current time)
- updated_at (timestamp, default to current time)

CustomOrderFiles Table
======================
- file_id (PK, UUID)
- order_id (FK, UUID, references Orders(order_id), onDelete CASCADE)
- file_url (string, not null)
- created_at (timestamp, default to current time)
- updated_at (timestamp, default to current time)
